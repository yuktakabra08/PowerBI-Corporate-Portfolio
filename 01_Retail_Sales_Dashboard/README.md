# Retail Sales Analytics Dashboard

## Business Problem
Analyze retail sales performance across products, regions, and time.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- SQL

## Key Features
- Star Schema Data Model
- Time Intelligence (YTD, YoY, MoM)
- Product & Region Performance Analysis
- Executive KPI View

## Files
- PBIX: Final Power BI report
- ETL_Documentation.md: Data cleaning & transformation steps
- DAX_Measures.md: All measures with explanation
- Model.png: Star schema diagram
