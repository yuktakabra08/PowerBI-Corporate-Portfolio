# 02 - SQL for Power BI

Objective: Become job-ready in SQL used by BI teams.

Topics:
1. SELECT, WHERE, ORDER BY
2. JOINS (INNER, LEFT, RIGHT, FULL)
3. GROUP BY, HAVING
4. Subqueries
5. CTEs (WITH clause)
6. Window Functions (ROW_NUMBER, RANK, LAG)
7. Views for reporting layer
8. Performance tuning for BI queries

Outcome:
You should be able to prepare reporting tables and connect them to Power BI.
